# T3D Modbus Speed Control — Справочник

## Железо

- Привод: HLTNC T3D-L20A-RABF
- Мотор: 80AST-A1C04025
- Связь: Modbus RTU / RS-485
- Порт: `/dev/ttyUSB0`, 19200 бод, 8E1, Slave ID=1

## Параметры для режима скорости через Modbus

| Параметр | Значение | Описание |
|----------|----------|----------|
| P-004    | 1        | Режим управления скоростью |
| P-025    | 1        | Источник скорости: внутренние пресеты P-137..P-144 |
| P-098    | 1        | Форсированное включение серво (без SON пина) |
| P-100    | 10       | DI1 функция = SP1 (выбор пресета 1) |
| P-137    | скорость | Скорость пресета 1, об/мин (отрицательное = реверс) |

Применить всё одной командой — кнопка **F** в TUI.

## Пин DI1 для активации SP1

Без активного DI1 (SP1) привод выдаёт скорость 0 даже при записанном P-137.

**Решение:** перемычка на CN1 между пин 31 и пин 16:

```
CN1 (DB44)
  pin 31  COM-  ────┐
                    │  перемычка ~10 см провода
  pin 16  DI1   ────┘
```

- Pin 31 = COM- (минус внутреннего 24В)
- Pin 16 = DI1 (цифровой вход 1)
- Внутри DI цепи стоит резистор 2кОм → ток ~11мА → безопасно

## Проверка без проводов (тест Fn-20)

Управление скоростью с кнопок панели привода (без Modbus и без DI):

```
◄  →  A-  →  SET  →  A-Fn  →  SET  →  Fn 0
→  ▼ (держать 3с)  →  Fn -20  →  SET (держать 3с)
→  r. 0  →  ▲ чтобы задать скорость
```

P-098 должен быть 1. Выход: ◄

## Modbus функциональные коды

| FC  | Описание | Примечание |
|-----|----------|-----------|
| 03H | Чтение параметров (FC03) | До 8 за раз |
| 04H | Чтение статуса (FC04) | До 8 за раз |
| 06H | Запись одного параметра | |
| 10H | Запись нескольких параметров | До 10 за раз |
| 41H | Сохранить EEPROM | Ждать 5с после |
| 42H | Серво вкл/выкл | 0x55=вкл, 0xAA=выкл |

## Адреса статуса FC04 (0x0000+)

| Адрес | Описание |
|-------|----------|
| 0x00  | Скорость мотора, об/мин (int16) |
| 0x09  | Момент, % от номинала |
| 0x0B  | Ток, А×10 |
| 0x0E  | Задание скорости, об/мин |
| 0x12  | Состояние DI (бит0=DI1 ... бит7=DI8) |
| 0x13  | Состояние DO |
| 0x1A  | Код аварии (0=норма) |
| 0x26  | Температура радиатора, °C |
| 0x28  | Напряжение шины DC, В |

## Запуск

```bash
# TUI (рекомендуется)
go run cmd/tui/main.go -slave 1 -port /dev/ttyUSB0

# CLI
go run cmd/main.go -slave 1 -port /dev/ttyUSB0 -read-status
go run cmd/main.go -slave 1 -port /dev/ttyUSB0 -write-param -addr 137 -val 300
```
