# Подключение RS-485: топология и монтаж

## Короткие расстояния (≤ 2 метра)

На расстоянии до 2 метров при скорости ≤ 115200 бод — почти любой кабель работает:

- Обычный patch cord (cat5/cat6)
- Плоский монтажный провод
- ШВВП

**Почему:** время бита при 115200 бод ≈ 8.7 мкс. Сигнал пробегает 2 метра за ~10 нс — рефлексии успевают затухнуть прежде чем драйвер считает следующий бит. Терминаторы на такой длине необязательны, хотя и не вредят.

**Важно даже на коротких расстояниях:** не прокладывать рядом с силовыми кабелями серводрайверов.

---

## Длинные расстояния (> 2 метров)

На длинных трассах — рефлексии, затухание и наводки становятся критичными. Всё нижеследующее обязательно.

### Топология: только гирлянда (daisy-chain)

```
                      120Ω
                        │
USB-RS485 ─── Drive1 ─── Drive2 ─── Drive3 ─── Drive4
    │                                               │
  120Ω                                           120Ω
(начало)                                         (конец)
```

- Каждый привод имеет два RS-485 порта: **входной** и **выходной** (на T3D: CN3 вход, CN4 выход).
- Соединяешь: адаптер → CN3(Drive1) → CN4(Drive1) → CN3(Drive2) → ... → CN4(Drive4).
- CN4 последнего привода остаётся свободным — туда ставится терминирующий резистор.

**Нельзя использовать звезду** (один адаптер — несколько лучей к приводам): рефлексии на отводах гарантируют нестабильность.

---

### Кабель

| Характеристика | Значение |
|---|---|
| Тип | Витая пара с экраном (STP / FTP) |
| Характеристическое сопротивление | **120 Ом** |
| Сечение жилы | 0.20–0.35 мм² (AWG 24–22) |
| Экран | обязателен на длинах > 5м |

**Подходящие марки:**
- Belden 9842 / 3105A (международный стандарт)
- КСПЭВ 2×0.5 (отечественный, дешевле)
- Любой кабель с маркировкой "RS-485", "120Ω twisted pair"

**Не подходит:**
- cat5/cat6 patch cord (100Ом, нет экрана или экран не для этого)
- ШВВП, ВВГ, монтажный провод — нет скрутки
- Коаксиальный кабель

---

### Терминирующие резисторы 120 Ом

Резистор ставится **между линиями A и B** на обоих концах шины.

#### Начало шины — у USB-RS485 адаптера

Многие адаптеры имеют встроенный терминатор (джампер или DIP-переключатель). Проверь документацию на адаптер и включи его.

Если нет встроенного — припаяй резистор 120 Ом между клеммами A и B на клеммной колодке адаптера.

#### Конец шины — CN4 последнего привода

**Если CN4 — клеммная колодка (винтики):**
```
CN4:  [A] [B] [GND]
       │   │
      120Ω (ставишь ножками прямо в клеммы A и B)
```

**Если CN4 — разъём RJ-45:**

Делаешь заглушку-терминатор:
1. Возьми пустой коннектор RJ-45
2. Найди в мануале T3D какие пины RJ-45 это A и B для CN4
3. Припаяй резистор 120 Ом между этими двумя пинами внутри коннектора
4. Вставь в CN4

Такая заглушка — стандартное решение, называется "termination plug".

### Экранирование и заземление

- Экран кабеля подключай к земле **только с одной стороны** — со стороны USB-адаптера (master)
- На стороне Drive4 (конец шины) — экран оставь висеть или подключи через конденсатор 10 нФ на корпус привода
- **Почему не с обеих сторон:** земляная петля между адаптером и приводом — антенна для помех от инвертора

```
USB-адаптер                              Drive4
  [экран] ── к GND шасси         [экран] ── не подключать
                                          (или 10нФ на корпус)
```

---

### Прокладка кабеля

- Кабель RS-485 и силовые кабели (питание моторов, кабели энкодеров с питанием) — **в разные кабельные каналы**
- Минимальное расстояние при параллельной прокладке: 20–30 см
- Если кабель всё же пересекается с силовым — пересечение под углом **90°**, не параллельно
- Серводрайвер при переключении генерирует ВЧ-импульсы (dV/dt), которые наводятся в соседние проводники

### Параметры для данного проекта

| Параметр | Значение |
|---|---|
| Скорость | 115200 бод |
| Приводов | 4 (slave ID 1–4) |
| Топология | гирлянда, адаптер → M1 → M2 → M3 → M4 |
| Расчётная длина шины | ~80 м (4 сегмента по ~20 м) |
| Терминаторы | у USB-адаптера и у M4 (CN4) |
| Макс. допустимая длина при 115200 бод | ~1000 м (теоретически), 200–300 м практически |

При 80 метрах и 115200 бод — никаких проблем с длиной нет. Основные риски: качество кабеля, отсутствие терминаторов, наводки от силовых цепей.

---

### Быстрая проверка при проблемах

1. **Постоянные таймауты** → скорее всего нет терминаторов или кабель не витая пара
2. **Работает, но иногда теряет пакеты** → наводки от силовых кабелей; разнести трассы
3. **Один привод не отвечает** → проверь slave ID (P-179 в настройках), полярность A/B, физический обрыв
4. **Всё работает на 2м, не работает на 20м** → добавь терминаторы и смени кабель на STP 120Ом
5. **Помогает только замедление до 19200 бод** → рефлексии из-за отсутствия терминаторов или звёздной топологии
